# Tomca For Mac

## 安装

```
brew install tomcat@8
```

安装地址：

```
/usr/local/opt/tomcat\@8/ ->  /usr/local/Cellar/tomcat\@8/8.5.41/
```

启动服务
 
```
brew tap homebrew/services
brew services start tomcat
```
