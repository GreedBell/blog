# Java 服务

* Jetty 是一个纯粹的基于 Java 的网页服务器和 Java Servlet 容器。
* Gretty, Gradle 插件，它不仅支持 Jetty servlet 容器,还支持 Tomcat servlet 等多种容器.配置也相对简单。
