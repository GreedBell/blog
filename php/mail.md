# PHP 使用 Mail

## 使用 sendmail

Linux 下 可以使用 sendmail 工具发邮件

### 配置 sendmail

`php.ini` 配置 `sendmail-path` 为 sendmail 路径

参考 <http://php.net/manual/zh/mail.configuration.php#ini.sendmail-path>

### 发送邮件

使用 [Mail 函数](http://php.net/manual/zh/function.mail.php) 发送邮件
