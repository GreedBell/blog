# Debug

Debug 界面各个图标的意思，参考 [Debugging with Xcode](https://developer.apple.com/library/archive/documentation/DeveloperTools/Conceptual/debugging_with_xcode/chapters/debugging_tools.html)

```
L = Local variable
A = Argument
S = Static variable
V = global Variable
R = register
V = instance Variable
E = Expression
```
